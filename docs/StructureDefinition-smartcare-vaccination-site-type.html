Resource Profile: Vaccination site type in Smartcare

Official URL: http://openhie.org/fhir/zambia-immunizations/StructureDefinition/smartcare-vaccination-site-type Version: 1.0.0
Active as of 2024-08-14 Computable Name: SmartcareVaccinationSiteType
